# Plotly Dash Example

This is a simple example of a web page built with Plotly and Dash.

## Setup

1. Install the required packages:
   ```
   pip install -r requirements.txt
   ```

## Running the App

Run the app with:
```
python app.py
```

The app will start a local server. Open your browser and go to `http://127.0.0.1:8050/` to view the page.

## What it does

This example creates a simple Dash application with a Plotly graph showing a sine wave.
